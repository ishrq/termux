# Dotfiles for Termux

## Dependencies

[GNU Stow](https://www.gnu.org/software/stow/){:target="_blank"}

## Instructions

```
# Clone repository
git clone git@github.com:ishrq/termux.git ~/.dotfiles/

# Change directory
cd ~/.dotfiles/

# Copy configs with Stow
stow -S <package-name>  #e.g. `stow -S fish`

```
